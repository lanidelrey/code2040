code2040
========

API Challenge for Code2040 (Leilani Reyes)

0. Registration (email, github) to retrieve token
1. Reversal of given string
2. Finding instance of string in an array of strings (needle in haystack)
3. Creating an array of strings (from initial array of strings) that do not include the given prefix
4. Adding the given interval to the given timestamp, ISO 8601 format

(All of the above challenges involve JSON handling)
